About Us

Built for used Tesla buyers, sellers, and dealers who want less noise.

Only Used Tesla is an independent Tesla-only marketplace and software project built by Adam Qureshi in New York City. Since 2017, the goal has been simple: make used Tesla shopping, selling, and dealer marketing more focused, more mobile-friendly, and easier to trust.

What Only Used Tesla is building

Only Used Tesla is part marketplace, part software shop, and part performance marketing system for the used Tesla market.

Tesla-only focus

A dedicated experience for used Tesla inventory instead of a generic marketplace full of unrelated vehicles.

Mobile-first by default

Most users are on phones, so the experience is designed around scanning, tapping, and getting to the point quickly.

Built for dealers and private sellers

The platform supports both consumer listings and dealership inventory with clearer Tesla-specific positioning.

Independent software shop

This project is intentionally lean. Small teams can still build useful software, fast pages, and focused products.
