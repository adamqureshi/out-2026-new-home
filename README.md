# Only Used Tesla FAQ page

I chose **FAQ** for the file name and footer link because it is the clearest label for search and navigation.
The visible headline on the page still says **Common questions.**

## Files

- `faq.html` → upload this into the same folder as `index.html`, `about.html`, `contact.html`, `dealer.html`, and `out-match.html`
- `footer-link-snippet.html` → copy this line into the footer on your other pages

## Fast install

1. Upload `faq.html` into your `/out-2026-new-home/` folder.
2. Add the FAQ footer link to your shared footer or to each page footer.
3. Open the new page at `faq.html` and test it on mobile.
4. If you use Search Console, request indexing after upload.

## How to add more questions later

Inside `faq.html`, duplicate one full block that starts like this:

```html
<details class="faq-item">
```

Then replace the question text and answer text.

You do **not** need to hand-write separate FAQ schema.
The page automatically builds the FAQ structured data from the visible questions on the page.
